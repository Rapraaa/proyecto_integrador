para la construcción de la webapp llamada simulador de entrevistas, hemos usado el framework backend de nodeJS escrito en typescript llamado nestJS
Recordar que typescript es un superconjunto de jabascript.
La página de inicio (home page) responderá la frase: He llegado a tu sitio web y te doy diez segundos para que me expliques qué haces y por qué debería elegirte a ti en lugar de a tu competencia. 
El proyecto a nivel técnico es un sitio web con páginas dinámicas con input del usuario y actualizaciones en tiempo real. El sistema será usado por estudiantes de todos los cemestres de la carrera de ingeniería en ciencias de la computación o carreras a fines como tecnología en desarrollo de software. Al finalizar La entrevista simulada se le proporcionará al usuario una calificación, con el fin de que el estudiante que quiere integrarse al entorno laboral adquiera mayor seguridad a la hora de enfrentarse tanto a la parte teórica como a la parte práctica de una entrevista laboral real (esta string es el valor de la clave description en simulador_entrevistas/package.json)
El patrón arquitectónico usado para la construcción de esta webapp, es conocido como modelo vista controlador, y abreviado como MVC.
